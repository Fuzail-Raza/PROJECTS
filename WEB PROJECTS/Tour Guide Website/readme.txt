Introduction
The Tour Guide Website is an online platform that allows users to book tours, choose their tour guide, and contact them via email. The website is built using HTML, CSS, JavaScript, and Bootstrap to create an interactive and visually appealing user experience.

******************************************************************************************************************************************************************************************************************************************************************
Features
The Tour Guide Website offers the following features:

Tour Booking: Users can browse through a variety of tours and select their preferred tour package.

Guide Selection: Users can choose a tour guide from a list of available guides associated with each tour.

Booking Confirmation: After selecting a tour and guide, users can confirm their booking and provide their contact details.

Email Notification: Upon successful booking, an email is automatically sent to the chosen tour guide with the user's contact information.

******************************************************************************************************************************************
Usage
To use the Tour Guide Website:

Clone or download this repository to your local machine.

Open the home page.php file in a web browser.

Browse through the available tours, select a tour, choose a guide, and provide your contact details.

Click the "Confirm Booking" button to complete the booking process.

An email will be automatically sent to the chosen tour guide with your contact information.

*********************************************************************************************************************************
Customization :
You can customize the Tour Guide Website according to your specific needs:

Modify the content of home page.html to update tour descriptions, guide information, and booking forms.

Adjust the styling by modifying the CSS files.

Customize the functionality by editing the JavaScript files.
