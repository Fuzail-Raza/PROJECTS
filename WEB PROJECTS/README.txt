Introduction
This project is a Cricket Info Website that provides various features for cricket enthusiasts. It allows users to view rankings, upcoming matches, buy team merchandise, and create user profiles. The website also includes an admin login for administrative tasks.

-----------------------------------------------------------------------------------------------------------------------------------------

Database Setup
Before running the project, make sure to set up the database by following these steps:

Locate the "DB" folder in the project directory.
Use the provided database file to import the database structure and tables into your preferred database management system (e.g., MySQL, PostgreSQL).
Import the data file to populate the tables with initial data.

Running the Project
To run the Cricket Info Website project, follow these steps:

Make sure you have the necessary software requirements installed, such as a web server (e.g., Apache), PHP, and a supported database management system.
Copy the project files to the root directory of your web server.
Ensure that the web server is running.

Open a web browser and navigate to the project's URL (e.g., http://localhost/cricket-info-website).
You should now see the homepage of the Cricket Info Website.
